2013년에 초판이 나온 책이다.

즉 최신 LTS인 Java 17은커녕 Java 8 관련 이야기도 없다. 그런데 놀라운 건 그럼에도 불구하고 읽을거리가 있었다는 점이다.

<!-- truncate -->

한 줄 요약하면 Java 애플리케이션과 JVM의 성능 튜닝의 콘셉트와 방향성을 잡아주는 책이다. 

커맨드나 옵션이 변경되었을진 몰라도 원리와 콘셉트는 예나 지금이나 크게 달라지지 않았다고 생각이 드는 게 신기하기도 하고 그렇다. 개념에 대해서 이해할 수 있을 정도로만 최소한의 설명을 하고 이야기하듯이 글이 적혀있는데 그렇다 보니 읽으면서 막히는 부분도 많이 없어서 좋았다.
