1. 내 풀이


```
class Solution:
    def reverse(self, x: int) -> int:
        x = int(str(x)[::-1]) if str(x)[-1] != "-" else int("-" + str(x)[::-1][:-1])
        return x if abs(x) < 2 ** 31 else 0
```
