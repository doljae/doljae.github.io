### 1\. 선형 탐색

```
class Solution:
    def maxProfit(self, prices: List[int]) -> int:
        min_val, profit = prices[0], 0
        for price in prices:
            min_val = min(min_val, price)
            if profit < price - min_val:
                profit = price - min_val
        return profit
```
