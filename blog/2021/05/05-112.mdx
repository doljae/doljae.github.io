### 1\. Stack을 사용한 풀이

```
class Solution:
    def reverseList(self, head: ListNode) -> ListNode:
        if not head:
            return None
        stack = []
        cur = head
        while cur:
            stack.append(cur)
            cur = cur.next
        new_head = stack.pop()
        cur = new_head
        while stack:
            cur.next = stack[-1]
            cur = cur.next
            stack.pop()
        cur.next = None
        return new_head
```

### 2\. Pointer를 사용한 (나름) 우아한 풀이

```
class Solution:
    def reverseList(self, head: ListNode) -> ListNode:
        if not head:
            return None
        rev = None
        cur = head
        while cur:
            rev, rev.next, cur = cur, rev, cur.next
        return rev
```
