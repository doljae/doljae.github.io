### 1\. Object id값과 집합을 이용한 풀이

```
class Solution:
    def hasCycle(self, head: ListNode) -> bool:
        id_set = set()
        cur = head
        while cur:
            if id(cur) not in id_set:
                id_set.add(id(cur))
            else:
                return True
            cur = cur.next
        return False
```

### 2\. Floyd의 순환 알고리즘

```
class Solution:
    def hasCycle(self, head: ListNode) -> bool:
        slow = fast = head
        while fast and fast.next:
            slow = slow.next
            fast = fast.next.next
            if slow == fast:
                return True
        return False
```
